🚀 DEPLOYMENT WORKFLOW

STEP 1: Prepare VPS
   □ Ubuntu 20.04+ or Debian 11+ (64-bit)
   □ At least 2GB RAM, 10GB disk space
   □ Root or sudo access
   □ Domain configured and DNS pointing to VPS

STEP 2: Download & Run Installation
   $ cd /home/tradezone
   $ sudo bash install.sh
   
   Installation includes:
   ✓ Node.js installation
   ✓ PostgreSQL database setup
   ✓ Apache2 web server configuration
   ✓ SSL certificate (Let's Encrypt - via certbot)
   ✓ Dummy test account creation
   ✓ Systemd service registration
   ✓ Firewall configuration
   
   Expected Duration: 5-15 minutes

STEP 3: Initial Configuration
   $ sudo nano /var/www/tradezone/backend/.env
   
   Update:
   • CORS_ORIGIN: https://yourdomain.com
   • Payment gateway credentials
   • Kraken API keys (optional)

STEP 4: First Login
   URL: https://yourdomain.com
   Email: admin@tradezone.local
   Password: Admin@123456
   
   ✅ You're now logged in as admin!

STEP 5: Change Default Credentials
   • Settings → Security → Change Password
   • Create strong new admin password
   • Keep password secure
   • Do NOT share with others

STEP 6: Verify Test User Account
   • Logout of admin
   • Login with test user: user@tradezone.local / User@123456
   • Test trading features
   • Verify all functionality works
   • Change password

STEP 7: Clean Up Demo Accounts
   • Delete: /var/www/tradezone/DUMMY_ACCOUNTS.txt
   • Delete test accounts from Admin Panel
   • Create real user accounts
